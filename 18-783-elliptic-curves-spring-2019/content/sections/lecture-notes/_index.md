Some of the theorems and algorithms presented in lecture are demonstrated using [Sage](http://sagemath.org/), an open-source computer algebra system with extensive support for computing with elliptic curves. You can download a copy of Sage to run on your own machine if you wish, or create an account for free on the [Collaborative Calculation in the Cloud](https://cloud.sagemath.com/?utm_source=sagemath.org&utm_medium=banner).

Please download the PDFs to view them with a PDF reader because some links lead to a non-OCW site that does not allow easy navigation back to the OCW site.
